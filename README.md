# 🍳 Ania Rozgotuje

> Portal recenzyjno-rozrywkowy testujący przepisy kulinarne z internetu.
> Gotujemy, oceniamy, nie żałujemy. Mema z szacunkiem.

**Tagline:** *Testujemy przepisy zamiast was — żebyście nie musieli.*

---

## 📌 Koncepcja

"Ania rozgotuje" to nie atak na oryginał — to niszowy, autentyczny kontent recenzyjny.
Anna Kowalska (nasza Ania) testuje przepisy z popularnych serwisów kulinarnych i szczerze mówi co z tego wyszło.

**Format:** rozrywka + recenzja = "efekt książula dla kuchni"
**Ton:** ciepły sarkazm, autentyczność, humor bez złośliwości
